# What I learned

Today I learned about Python lists and how to store multiple values in a single variable. I practiced list indexing, slicing, and useful list methods like append(), insert(), remove(), and pop(). I also learned how CRUD (Create, Read, Update, Delete) operations work.

Using these concepts, I built my first Student Management System where I can add, view, search, update, and delete student records using a menu-driven program.
