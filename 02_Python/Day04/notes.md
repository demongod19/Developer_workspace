## elif

Meaning:
elif means "Else If".

Purpose:
It checks another condition only if the previous condition is False.

Syntax:

if condition:
    statement

elif condition:
    statement

else:
    statement

Important Points

- Python checks conditions from top to bottom.
- Only the first True condition executes.
- Remaining conditions are skipped.
- Multiple elif statements are allowed.

Real Life Uses

- Student grading
- ATM menu
- Salary bonus
- Weather system
- Ticket booking
