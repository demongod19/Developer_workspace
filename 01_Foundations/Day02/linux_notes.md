# Linux Terminal Notes - Day 2

## 1. pwd

**Meaning:** Print Working Directory

**Purpose:** Displays the current directory in which the terminal is working.

Example:

```bash
pwd
```

Output:

```
D:\Developer_workspace\01_Foundations\Day02
```

---

## 2. ls

Lists all files and folders in the current directory.

```bash
ls
```

---

## 3. cd

Changes the current directory.

Example:

```bash
cd Practice
```

---

## 4. cd ..

Moves one level up to the parent directory.

Example:

```bash
cd ..
```

---

## 5. cd .

Represents the current directory.

Running `cd .` keeps you in the same location.
